# 特徴埋め込みを持つ Vision Transformer を用いたディープフェイク検知に関する研究
Author: Rakutaro Sorahata

---

## 📘 概要
本研究では、Vision Transformer（ViT）に学習済みの畳み込み層（CNN）を組み合わせた特徴埋め込み構造を導入し、ディープフェイク画像の検知精度を向上させる手法を提案しました。  
既存のViTモデルでは過学習やデータ依存性が強く、特に小規模データセットでは精度が不安定になる問題がありました。  
本研究のモデルでは、CNNによる事前特徴抽出を組み合わせることで、より安定した学習挙動を実現しています。

---

## 🎯 目的
ディープフェイク画像を、学習済み畳み込み層を持つVision Transformerを用いて検知し、従来のViT単体モデルに対して性能の安定化および精度向上を図る。

---

## 🧠 使用技術
- **フレームワーク**: TensorFlow  
- **モデル構成**: CNN + Vision Transformer  
- **主なライブラリ**: NumPy, OpenCV, TensorFlow, Keras, Matplotlib  
- **開発環境**: Google Colaboratory  

---

## 💾 データセット
Meta社（旧Facebook社）が公開している **Deepfake Detection Challenge (DFDC)** の動画から、独自にフレームを切り出して構成した自作データセットを使用しました。  
※ライセンスの都合上、データセットは本リポジトリには含まれていません。

---

## 📈 実験結果
- Meta社が主催した **DFDCコンペティションの最高記録を更新**。  
- 先行研究における ViT 単体モデルと比較して、**シード値の揺らぎによる過学習の頻度が低下**。  
- CNNによる事前特徴抽出により、入力のロバスト性と学習安定性が向上。
